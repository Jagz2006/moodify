# Moodify 🎵
> music for your moment

Moodify is a web app that creates personalized Spotify playlists based on your mood — 
pulling songs only from your own liked tracks and library, not random internet picks.

Pick a mood (Happy, Sad, Energetic, Chill, Romantic, or Focus), and Moodify uses 
Spotify's Recommendations API seeded with your own songs to find the perfect matches. 
The playlist gets saved directly to your Spotify account in seconds.

## Features
- 🔐 Secure Spotify login via OAuth 2.0 (PKCE — no backend needed)
- 🎧 Pulls from your personal liked songs and playlists only
- 😄 6 moods mapped to Spotify's audio parameters (valence, energy, tempo)
- ⚡ Auto-creates and saves the playlist to your Spotify account
- 📱 Fully responsive — works on mobile and desktop
- 🌙 Dark mode UI with mood-reactive color themes
